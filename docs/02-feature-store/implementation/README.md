## 🎯 학습 순서

1. **기본 개념** (2.1-2.2): 피처 엔지니어링과 파이프라인 기초
2. **품질 관리** (2.3-2.4): 검증, 테스팅, 메타데이터 관리
3. **고급 구현** (2.5-2.7): 실제 피처 스토어 구축과 툴 활용

## 📋 전제 조건

- Python 3.8+ 환경
- 기본적인 pandas, scikit-learn 지식
- Docker (선택사항, Feast 사용 시)
